Secret Santa frontend

Файлы:
- index.html — регистрация и отправка пожелания
- wish.html — просмотр пожелания друга
- styles.css — общий стиль
- app.js — логика главной страницы
- wish-page.js — логика страницы просмотра пожелания

Что может потребоваться поправить:
1. Если фронт и бэкенд работают на одном домене/порту, ничего менять не нужно.
2. Если бэкенд работает отдельно, укажи базовый URL в файлах app.js и wish-page.js:
   const API_BASE = "http://localhost:5000";

Используемые эндпоинты:
- POST /api/santa/register
- POST /api/santa/wish
- GET  /api/santa/wish/{name}

Важно:
- Если фронт открыт с одного origin, а API на другом, на сервере должен быть включен CORS.
